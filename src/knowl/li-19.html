Item 2
The number of times that an element appears in an enumeration doesn't affect a set. For example, if \(A = \{1, 5, 3, 5\}\) and \(B = \{1, 5, 3\}\), then \(A = B\). Warning to readers of other texts: Some books introduce the concept of a multiset, in which the number of occurrences of an element matters.
