Example1.2.16Some Symmetric Differences
  1. Let \(A = \{1, 3, 8\}\) and \(B = \{2, 4, 8\}\). Then \(A \oplus B = \{1, 2, 3, 4\}\).
  2. \(A \oplus 0 = A\) and \(A \oplus A = \emptyset\) for any set \(A\).
  3. \(\mathbb{R} \oplus \mathbb{Q}\) is the set of irrational numbers.
