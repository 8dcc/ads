We will use the notation \(\mathbb{Z}_n\) to denote the set \(\{0, 1, 2,. . ., n-1\}\text{.}\) One interpretation of this set is that each element is a representative of its equivalence class with respect to congruence modula \(n\text{.}\) For example, if \(n=7\text{,}\) the number \(1\) in \(\mathbb{Z}_7\) really represents all numbers in \([1]={1+7k : k\in \mathbb{Z}}\text{.}\) In doing modular arithmetic, we can temporarily replace elements of \(\mathbb{Z}_n\) with other elements in their equivalence class modulo \(n\text{.}\)
