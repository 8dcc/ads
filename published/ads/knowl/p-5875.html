  1. Make a list of the different cycle structures that occur in \(A_5\) and how many elements have those structures.
  2. Within each set of permutations with different cycle structures, identify which subsets are closed with respect to the conjugation operation. With this you will have a partition of \(A_5\) into conjugate classes where for each class, \(C\text{,}\) \(f, g\in C\) if and only if \(\exists \phi \in A_5\) such that \(\phi ^{-1}\circ f\circ \phi = g\text{.}\)
  3. Use the fact that a normal subgroup of \(A_5\) needs to be a union of conjugate classes and verify that no such union exists.
