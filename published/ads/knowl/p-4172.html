For example, in \(\mathbb{U}_{2048}\text{,}\) if we want the muliplicative inverse of \(1001\text{,}\) we run the Extended Euclidean Algorithm and find that
\begin{equation*} \gcd(2048,1001) = 1 =457\cdot 2048+ (-935)\cdot 1001 \end{equation*}
Thus, the multiplicative inverse of 1001 is \(2048-935 = 1113\text{.}\) See the SageMath Note below to see how to run the Extended Euclidean Algorithm.
