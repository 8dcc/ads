Let \(A\) be an \(n \times n\) matrix. If a matrix \(B\) can be found such that \(A B = I\text{,}\) then \(B A = I\text{,}\) so that \(B = A^{-1}\text{.}\) In fact, to find \(A^{-1}\text{,}\) we need only find a matrix \(B\) that satisfies one of the two conditions \(A B = I\) or \(B A = I\text{.}\)
