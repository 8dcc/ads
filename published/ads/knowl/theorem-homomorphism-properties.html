Theorem 15.4.14. Group Homomorphism Properties.

If \(\theta: G \rightarrow G'\) is a homomorphism, then:
  1. \(\theta(e) =\theta(\textrm{the identity of } G) = \textrm{the identity of } G' = e'\text{.}\)
  2. \(\theta\left(a ^{-1}\right) = \theta(a)^{-1}\) for all \(a \in G\text{.}\)
  3. If \(H \leq G\text{,}\) then \(\theta(H) = \{\theta(h) | h\in H\}\leq G'\text{.}\)
