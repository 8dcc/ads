Exercise 16.

It was observed above that in doing modular arithmetic, one can replace an element of \(\mathbb{Z}_n\) with any other element of its equivalence class modulo \(n\text{.}\) For example, if one is computing \(452 \times_{461} 7\text{,}\) the alternative to multiplying \(452\) time \(7\) and then dividing by \(461\) to get the remainder in \(\mathbb{Z}_{461}\text{,}\) we can replace \(452\) with \(-9\) and get a product of \(-63\) which is conguent of \(398\text{.}\) Use this “trick” to compute the following without the use of a calculator.
  1. \(\displaystyle 898 \times_{1001} 998\)
  2. \(\displaystyle 77^{10} \mod{81} \)
  3. The solution to \(196 \times_{197} x = 120\)
