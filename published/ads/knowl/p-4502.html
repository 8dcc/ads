Given a system of equations involving real variables \(x_1\text{,}\) \(x_2, \ldots \text{,}\) \(x_n\text{,}\) the solution set of the system is the set of \(n\)-tuples in \(\mathbb{R}^n\text{,}\) \(\left(a_1, a_2, \ldots ,a_n\right)\) such that the substitutions \(x_1= a_1\text{,}\) \(x_2= a_2, \ldots\text{,}\) \(x_n= a_n\) make all the equations true.
