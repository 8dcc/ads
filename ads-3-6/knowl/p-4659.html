We begin by developing a helpful technique for computing \(A^m\text{,}\) \(m > 1\text{.}\) If \(A\) can be diagonalized, then there is a matrix \(P\) such that \(P^{-1}A P = D\text{,}\) where \(D\) is a diagonal matrix and

\begin{gather} A^m= P D^m P^{-1} \textrm{ for all } m\geq 1\tag{12.5.1} \end{gather}
