Item 15.4.3.10.b.

Within each set of permutations with different cycle structures, identify which subsets are closed with respect to the conjugation operation. With this you will have a partition of \(A_5\) into conjugate classes. where for each class \(C\text{,}\) \(f, g\in C\) if and only if \(\exists \phi \in A_5\) such that \(\phi ^{-1}\circ f\circ \phi = g\text{.}\)
