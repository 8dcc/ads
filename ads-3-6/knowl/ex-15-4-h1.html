Example 15.4.13. Decreasing modularity.

Define \(\alpha:\mathbb{Z}_6\to \mathbb{Z}_3\) by \(\alpha(n)=n \textrm{ mod } 3\text{.}\) Therefore, \(\alpha(0) = 0\text{,}\) \(\alpha(1) = 1\text{,}\) \(\alpha(2) = 2\text{,}\) \(\alpha(3) =1 + 1 + 1=0\text{,}\) \(\alpha(4) = 1\text{,}\) and \(\alpha(5) = 2\text{.}\) If \(n, m \in \mathbb{Z}_6\text{.}\) We could actually show that \(\alpha\) is a homomorphism by checking all \(6^2=36\) different cases for the formula

\begin{gather} \alpha(n +_6 m) = \alpha(n)+_3\alpha(m)\tag{15.4.1} \end{gather}

but we will use a line of reasoning that generalizes. We have already encountered the Chinese Remainder Theorem, which implies that the function \(\beta: \mathbb{Z}_6\to \mathbb{Z}_3 \times \mathbb{Z}_2\) defined by \(\beta(n)=(n\textrm{ mod } 3, n\textrm{ mod } 2)\text{.}\) We need only observe that equating the first coordinates of both sides of the equation

\begin{gather} \beta(n +_6 m) = \beta(n)+_3\beta(m)\tag{15.4.2} \end{gather}

gives us precisely the homomorphism property.
