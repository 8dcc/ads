Given a matrix equation \(A x = b\text{,}\) where A is \(n \times m\text{,}\) let \(C\) be the augmented matrix \([A | b]\text{.}\) The process of row-reducing to echelon form involves performing the following algorithm where \(C[i]\) is the \(i^{\textrm{th}}\) row of \(C\text{.}\)

  1. i = 1

  2. j = 1

  3. while i <= n and j <= m):

    1. maxi=i

    2. for k = i+1 to n:

      \(\quad\)if abs(C[k,j])>abs(C[maxi,j]): then maxi=k

    3. if C[maxi,j] != 0 then:

      1. exchange rows i and maxi

      2. divide each entry in row i by C[i,j]

      3. for u = i+1 to n:

        \(\quad\)subtract C[u,j]*C[i] from C[u]

      4. i = i+1

    4. j=j+1
