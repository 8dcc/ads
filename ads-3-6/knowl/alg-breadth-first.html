Algorithm 9.3.8. Breadth-first Search.

A broadcasting algorithm for finding a path between vertex \(i\) and vertex \(j\) of a graph having \(n\) vertices. Each item \(V_k\) of a list \(V=\left\{V_1, V_2, \ldots , V_n\right\}\text{,}\) consist of a Boolean field \(V_k.\text{found}\) and an integer field \(V_k.\text{from}\text{.}\) The sets \(D_1\text{,}\) \(D_2, \dots\text{,}\) called depth sets, have the property that if \(k \in D_r\text{,}\) then the shortest path from vertex \(i\) to vertex \(k\) is of length \(r\text{.}\) In Step 5, a stack is used to put the vertex list for the path from the vertex \(i\) to vertex \(j\) in the proper order.

  1. Set the value \(V_k.\text{found}\) equal to False, \(k = 1, 2, \dots , n\)

  2. \(r = 0\)

  3. \(D_0= \{i\}\)

  4. while \((\neg V_j.\text{found}\)) and \(\left(D_r\right.\neq \emptyset )\)

    • \(D_{r+1}=\emptyset\)

    • for each k in \(D_r\text{:}\)

      \(\quad\)for each edge (k,t):

      \(\quad \quad \)If \(V_t.\text{found}\) == False:

      \(\quad \quad \quad \)\(V_t.\text{found}=\text{True}\)

      \(\quad \quad \quad \)\(V_t.\text{from} = k\)

      \(\quad \quad \quad \)\(D_{r+1}=D_{r+1}\cup \{t\}\)

    • \(r = r + 1\)

  5. if \(V_j.\text{found}\text{:}\)

    • \(S = Empty Stack\)

    • \(k=j\)

    • while \(V_k.\text{from} \neq i\text{:}\)

      \(\quad\)Push \(k\) onto \(S\)

      \(\quad\)\(k = V_k.\text{from}\)
