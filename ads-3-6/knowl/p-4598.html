In doing Example 12.4.8, the given \(3\times 3\) matrix \(A\) produced only two, not three, distinct eigenvalues, yet we were still able to diagonalize \(A\text{.}\) The reason we were able to do so was because we were able to find three linearly independent eigenvectors. Again, the main idea is to produce a matrix \(P\) that does the diagonalizing. If \(A\) is an \(n \times n\) matrix, \(P\) will be an \(n\times n\) matrix, and its \(n\) columns must be linearly independent eigenvectors.
