Item b.

The power set of \(\mathbb{Z}\text{,}\) \(\mathcal{P}(\mathbb{Z})\text{,}\) over union is a monoid with identity \(\emptyset\text{.}\) If \(S=\{\{1\},\{2\},\{3\}\}\text{,}\) then \(\langle S \rangle\) is the power set of \(\{1,2,3\}\text{.}\) If \(S=\{\{n\}:n\in \mathbb{Z}\},\) then \(\langle S\rangle\) is the set of finite subsets of the integers.
