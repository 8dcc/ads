Item c.

The set of \(n\times n\) matrices over the integers, \(M_n(\mathbb{Z})\text{,}\) \(n\geq 2\text{,}\) with matrix multiplication, is a monoid. This follows from the fact that matrix multiplication is associative and has an identity, \(I_n\text{.}\) This is an example of a noncommutative monoid since there are matrices, \(A\) and \(B\text{,}\) for which \(A B \neq B A\text{.}\)
