Example 15.3.13. Some Compositions.

  1. \((1, 2, 3, 4)(1, 2, 3, 4) = (1, 3)(2, 4)\)

  2. \((1, 4)(1, 3)(1, 2) = (1, 2, 3, 4)\text{.}\)

Notice that cyclic notation does not indicate the set which is being permuted. The examples above could be in \(S_5\text{,}\) where the image of 5 is 5. This ambiguity is usually overcome by making the context clear at the start of a discussion.
