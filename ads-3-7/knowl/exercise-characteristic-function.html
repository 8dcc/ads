Exercise 2.

Let \(A\) be a set and let \(S\) be any subset of \(A\text{.}\) Let \(\chi_S: A\to \{0,1\}\) be defined by

\begin{equation*} \chi_S(x)= \left\{ \begin{array}{cc} 1 & \textrm{if } x\in S \\ 0 & \textrm{if } x\notin S \\ \end{array} \right. \end{equation*}

The function \(\chi_S\) is called the characteristic function of \(S\text{.}\)

  1. If \(A = \{a, b, c\}\) and \(S = \{a, b\}\text{,}\) list the elements of \(\chi_S\) .

  2. If \(A = \{a, b, c, d, e\}\) and \(S = \{a, c, e\}\text{,}\) list the elements of \(\chi_S\text{.}\)

  3. If \(A = \{a, b, c\}\text{,}\) what are \(\chi_{\emptyset}\) and \(\chi_A\text{?}\)
